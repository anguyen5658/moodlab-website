09 — DEVICE INPUT METHOD

📱 Chọn Kiểu Tương Tác Device

User chọn input method + behavior mode. Setting này là GLOBAL — áp dụng xuyên suốt cả Live và Arena. Persist qua sessions.

9.1 — 4 Loại Input

💨

Puff

MIC detect airflow + Heating ON → tiêu hao extract. Default cho trải nghiệm đầy đủ.

🌀

Dry Puff

MIC detect + Heating OFF → KHÔNG tiêu hao extract. Toggle trong Settings.

🔘

Button Press

Nút vật lý trên device. Signal qua BLE. Press / hold / release.

BỔ SUNG
👆

On-screen Tap

Chạm trên app. CHỈ BỔ SUNG — không thay thế device hoàn toàn.

⚠️
NGUYÊN TẮC INPUT — KINH DOANH

Mọi game và mọi tương tác (Puff Sync, Puff Wave, Challenges, Halftime Games) đều phải có ít nhất 1 bước cần device input (Puff, Dry Puff, hoặc Button). On-screen Tap chỉ là bổ sung, KHÔNG đủ để chơi mà không có device. Device drives revenue.

9.2 — Device Input Selector (UI)

📍 Vị trí Header Button

Góc trên bên phải, cạnh Notification + Search icons. Hiện ở cả Explore và Watch screen. Icon thay đổi theo input đang chọn (💨/🌀/🔘). Green dot indicator khi device connected.

📋 Popup Panel (3 phần)

1. Input Method: 3 options với accent color riêng.
2. Behavior: Auto / Fixed / Ask Each Session.
3. Device Status: Tên device, battery %, BLE, Connect/Reconnect.

9.3 — Behavior Modes

⚡

Auto

System tự chọn input phù hợp theo context. Arena game → Puff. Chill session → Dry Puff.

📌

Fixed

Luôn dùng method đã chọn. Không hỏi lại. Consistent experience.

❓

Ask Each Session

Hỏi user mỗi lần bắt đầu live/game mới. Linh hoạt nhất.

9.4 — Input Áp Dụng Xuyên Suốt

ContextẢnh hưởngVí dụ
Live — Puff SyncBubble icon + animation thay đổiDry Puff: bubble không có smoke. Button: pulse effect.
Live — Puff WaveCùng mechanic, khác visualPuff = smoke wave. Dry Puff = air wave. Button = tap wave.
Live — PUFF ButtonLabel + icon + color thay đổiPuff: 💨 pink. Dry Puff: 🌀 cyan. Button: 🔘 purple.
Arena GamesInput Map riêng per gameXem Arena Guideline Section 1.3 cho mapping chi tiết.
Halftime GamesKế thừa input từ Live sessionUser đang Dry Puff → Halftime game dùng Dry Puff.
💾
LƯU Ý CHO DEV — PERSIST STATE

Device Input là global state. Persist qua sessions (AsyncStorage / UserDefaults). Load last selected input + behavior khi mở app. Khi device disconnect mid-session → fallback On-screen Tap + warning popup. Khi reconnect → restore previous input method tự động.
